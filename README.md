# steam-free-asf
search for free to keep games and automatically redeem them using ArchiSteamFarm addlicense.

### usage
change config in script if needed and create a cron or task scheduler entry to run this script.
recommeneded that you dont run it more than once every 30 minutes as it can take a while

not responsible if your account is banned for using this as it may be a violation of the steam tos
